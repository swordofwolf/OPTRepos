Особенности Языка

Данный язык будет языком программирования с C подобным синтаксисом, содержащий некоторые особенности, 
которые нацелены на облегчение понимания, увелечения простоты написания программ, и читемость кода.
Список особенностей:
	*Одно вырожение в одной строке
	Пример:
			int q;
			int z;
			q = 0;
			z = 0;
			q = z + 12;
			z = q + 5;
	*Низкоуровневые опирации выполняются с помощью встроенных функций(операции работы со строками, математические, опреции работы с массивами)
	*Отсутствие указателей
			
	В языке Крабик (V)_ii_(V) не поддерживаются следующие конструкции(и им подобные)
			q + z; z - 12; 
			
			int q = z - 5 + 22; 
			
			for(int i = 0; i < 4; i++ for(;;));;
